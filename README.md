# EHIRCLE BLK Application

## 📌 Overview
The **EHIRCLE BLK Application** is a mobile solution designed to help users efficiently manage their personal or business inventory. It provides features for adding, updating, categorizing, and analyzing items through a simple and user-friendly dashboard.

---

## 🎯 Requirements Gathering Phase

### ✅ Functional Requirements
The application allows users to:
1. Add new items  
2. Update existing items  
3. View and search for items  
4. Delete items  
5. Create categories  
6. Update categories  
7. Categorize items  
8. Delete categories  
9. View and search for categories  
10. View analysis of items on a simple dashboard  

### ⚙️ Non-functional Requirements
The application will be developed using:
- **Flutter** → Frontend framework for cross-platform mobile development  
- **Firebase** → Backend solution for authentication, database, and cloud storage  

---

## 🛠️ Tech Stack
- **Frontend:** Flutter (Dart)  
- **Backend:** Firebase (Firestore, Authentication, Storage)
